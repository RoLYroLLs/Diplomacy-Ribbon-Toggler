# Diplomacy Ribbon Toggler

Sid Meier's Civilization VII mod to show/hide the diplomacy ribbon.

Ever wanted to toggle the diplomacy ribbon? Now you can! This mod allows you to show or hide the diplomacy ribbon at will, giving you a cleaner interface when you don't need it.

**Features**

[B]Diplomacy Ribbon:[/B] Show/Hide the ribbon at will.

**Hotkeys**

[B]Toggle:[/B] CTRL-R (default, can be customized)

**Saved Games**

This mod does not affect saved games.
